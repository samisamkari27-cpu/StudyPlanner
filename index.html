How the style is chosen (criteria)

      

        

          
1) Goal (what the exam rewards)

          
    
            
  • Memorization: prioritize retrieval practice + spaced review.
    • 
            
  • Understanding: prioritize self-explanation + “why/how” prompts + quick recall checks.
    • 
            
  • Application: prioritize worked examples → lots of practice → error log.
    • 
          

        


        

          
2) Time-to-exam (what’s realistic)

          
    
            
  • Short: cut scope, drill high-yield, test yourself frequently.
    • 
            
  • Medium: mix learning + consolidation, spaced sessions.
    • 
            
  • Long: build foundations first, then transfer practice.
    • 
          

        


        

          
3) Difficulty (how hard it is for you)

          
    
            
  • Easy: push to higher-level testing faster.
    • 
            
  • Medium: alternate learn → test → fix gaps.
    • 
            
  • Hard: start core concepts, reduce materials, track mistakes.
    •
